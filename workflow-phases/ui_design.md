# UI Design フェーズ

## subagent実行

このフェーズは**subagentとして実行**される。
**並列フェーズ (parallel_design)** の一部として実行される。

---

## 目的

ユーザーインターフェースの設計を行う。

## 入力

- タスクディレクトリ: `{workflowDir}`
- 要件: `requirements.md`
- 調査結果: `research.md`

## 実行手順

1. **画面構成の設計**
   - レイアウト構造
   - コンポーネント配置

2. **コンポーネント設計**
   - 各コンポーネントの責務
   - Props/State定義

3. **インタラクション設計**
   - ユーザー操作
   - フィードバック

4. **レスポンシブ設計**
   - ブレークポイント
   - モバイル対応
